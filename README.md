# Kerolos' Portfolio Website

This is my personal portfolio website where I showcase my previous work that I have published on GitHub using the GitHub API, and also receive messages on my personal email.

## Technologies Used

- React
- React-router-dom
- React-bootstrap
- Sass
- Formik
- Yup
- Axios

## Features

- Displaying information about my previous projects and adding links to access them.
- Providing the ability to contact me through the "Contact" section.

## Demo

You can view a live demo of my portfolio website at https://kerolos2000.github.io/portfolio/.
